## About

This Magisk module will set your device's DNS servers to:

* `89.233.43.71` Unicast server hosted at AS9167 in Copenhagen, Denmark.

* `91.239.100.100` [Anycast server](https://blog.uncensoreddns.org/dns-servers/) from multiple locations.

> ### ⚠️ Warning
>
> Your ISP **may block** all requests if you use custom DNS servers.

## Download

You can [download](https://github.com/Strappazzon/UncensoredDNS4Magisk/releases/latest) the latest version of this module from the Releases page.

## Installation

1. Open **Magisk Manager**

2. Swipe to the right to **open the menu**

3. Tap on **Modules** and then tap on the **+** (plus) button

4. Select the **ZIP file** containing this module

5. **Reboot**
